# Fast Message

## Fast Message – simple messenger on Ruby on Rails

## How to use:

```bash
git clone https://github.com/eliseigolushko-prog/FastMessage
cd FastMessage
bundle install
rails db:migrate
bin/rails assets:precompile
bin/rails server
```
